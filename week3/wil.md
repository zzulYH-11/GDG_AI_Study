회귀 모델의 성능을 표현할 때, 전체 데이터 변동성 중 모델이 잡아낸 비율을 나타내는 결정계수와 모델의 예측값이 실제값과 평균적으로 어느 정도 차이나는지 보여주는 평균 절대 오차가 있다는 것을 알았다. 또 회귀 알고리즘에는 직선 형태로 모델링하는 선형 회귀, 고차항의 특성을 포함하여 곡선 형태로 모델링하는 다항 회귀, 변수 종류(특성 수)가 2개 이상일 때 사용하는 다항 회귀가 있다는 것을 배웠다. 곡선 형태로 나타나는 다항 회귀, 다중 회귀도 사실은 선형 회귀라는 사실도 알았다. 회귀 모델에 데이터를 훈련시키다 보면 모델이 훈련 세트에 과하게 적합해져 일반성이 떨어지는 과대적합, 모델이 훈련 세트의 규칙을 찾아내지 못해 성능이 낮아지는 과소적합이 발생하기도 하고, 이를 적정 수준으로 맞추기 위해 계수의 크기를 조절하는 규제 방법을 사용한다는 것도 알았다. 또 특성공학과 저번에 배운 데이터 전처리의 차이가 무엇인지 궁금해졌는데, 데이터 전처리는 모델이 돌아갈 수 있게 하한선을 맞추는 느낌이고, 특성 공학은 모델이 정답을 더 잘 맞히도록 상한선을 높이는 느낌이라는 것을 배웠고 둘은 겹치는 부분도 있는 완전히 분리된 개념이 아니라는 것을 배웠다.
